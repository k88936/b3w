# b3w

## build

on linux
```shell
./gradlew build
```

 windows
```cmd
gradlew.bat build
```
## run
* recommended using idea run config: "application'
* or
    on linux
    ```shell
    ./gradlew run
    ```

    on windows
    ```cmd
    gradlew.bat run
    ```
## debug
idea run config as debug: "application"
